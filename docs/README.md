## Self-Build

**Clone Repo**

```
git clone https://github.com/CoolPlayLin/Gh-Robot

cd Gh-Robot
```

**Install Build**
```
pip install build
```

**Build**
```
python -m build
```

Checkout
>*You can choose to use the stable or development version*

>**Stable**
>```
>git checkout main
>```
>**Development**
>```
>git checkout Dev
>```

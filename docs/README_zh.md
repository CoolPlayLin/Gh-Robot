# 🥰 本地构建

### **克隆仓库**

```
git clone https://github.com/CoolPlayLin/Gh-Robot

cd Gh-Robot
```

### **安装`Build`工具**
```
pip install build
```

### **构建**
```
python -m build
```

### 切换版本
>*你可以通过这些命令来切换要构建的版本*

>**稳定版**
>```
>git checkout main
>```
>**开发版**
>```
>git checkout Dev
>```
